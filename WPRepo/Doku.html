JsonRepository<T> Klasse

Die JsonRepository<T>-Klasse ermöglicht das Speichern und Laden von Datenobjekten vom Typ T in JSON-Dateien.

Konstruktoren

  • JsonRepository(): Erstellt eine neue Instanz der JsonRepository<T>-Klasse.
  • JsonRepository(string filePath): Erstellt eine neue Instanz der JsonRepository<T>-Klasse mit dem angegebenen Dateipfad.

Eigenschaften

  • Instance: Statische Eigenschaft, die eine Singleton-Instanz der JsonRepository<T>-Klasse zurückgibt.
  • FilePath: Dateipfad, in dem die Daten gespeichert werden sollen.

Methoden

  • GetData(): Ruft die geladenen Datenobjekte vom Typ T ab.
  • AddData(T data): Fügt ein neues Datenobjekt vom Typ T hinzu.
  • Dispose(): Gibt Ressourcen frei und speichert die Daten in der JSON-Datei.

Verwendung der JsonRepository-Klasse

  1. Importieren Sie das WPRepo-Namespace.
  2. Erstellen Sie eine Instanz der JsonRepository<T>-Klasse und legen Sie bei Bedarf den Dateipfad fest.
  3. Verwenden Sie die GetData()-Methode, um die gespeicherten Daten abzurufen.
  4. Fügen Sie neue Daten hinzu und speichern Sie sie.
  5. Vergessen Sie nicht, die Dispose()-Methode aufzurufen, um die Ressourcen freizugeben und die Daten zu speichern, wenn Sie fertig sind.
