FileLogger Klasse

Die FileLogger-Klasse ermöglicht das Protokollieren von Nachrichten in einer Datei.

Eigenschaften

  • Instance: Statische Eigenschaft, die eine Singleton-Instanz der FileLogger-Klasse zurückgibt.

Methoden

  • Log(LogLevel level, string message): Protokolliert eine Nachricht mit dem angegebenen Log-Level.
  • Dispose(): Gibt Ressourcen frei und beendet die Protokollierung.

Verwendung des FileLogger

  1. Importieren Sie das WPLoggingLibrary-Namespace.
  2. Verwenden Sie die Instance-Eigenschaft, um auf die Singleton-Instanz des FileLogger zuzugreifen.
  3. Verwenden Sie die Log()-Methode, um Nachrichten zu protokollieren.
  4. Stellen Sie sicher, dass Sie die Dispose()-Methode aufrufen, um die Ressourcen freizugeben, wenn Sie den FileLogger nicht mehr benötigen.
